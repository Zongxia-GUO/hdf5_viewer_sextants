# 📤 导出功能对比指南

## 🎯 两套导出系统

HDF5 Viewer 现在提供 **两套导出功能**，各有优势：

### 1️⃣ **PyQtGraph 内置导出** (右键菜单)
- 来源：pyqtgraph 库自带
- 位置：在图像/曲线上右键 → Export

### 2️⃣ **HDF5 Viewer 导出** (Ctrl+E)
- 来源：我们新增的功能
- 位置：Export 菜单 → Export Current Dataset...

---

## 📊 功能对比表

| 特性 | PyQtGraph (右键) | HDF5 Viewer (Ctrl+E) |
|------|------------------|---------------------|
| **触发方式** | 右键点击图像/曲线 | Ctrl+E 或菜单 |
| **导出内容** | **当前显示的视图** | **原始数据** |
| **分辨率** | 屏幕显示分辨率 | 完整数据分辨率 ⭐ |
| **SVG 矢量图** | ✅ | ❌ |
| **PNG** | ✅ | ✅ |
| **JPEG** | ❌ | ✅ |
| **TIFF** | ❌ | ✅ (16-bit) ⭐ |
| **CSV** | ✅ (基本) | ✅ (增强) ⭐ |
| **MATLAB** | ✅ | ❌ |
| **列名支持** | ❌ | ✅ (结构化数组) ⭐ |
| **特殊值处理** | 基本 | ✅ (NaN/Inf) ⭐ |
| **快速访问** | ✅ | - |

## 🎨 使用场景推荐

### 场景 1: 论文插图（矢量图） 🌟

**推荐**: PyQtGraph 右键导出

```
需求: 高质量矢量图，可缩放不失真

步骤:
  1. 调整视图到满意的显示效果
  2. 右键图像 → Export
  3. 选择 "Scalable Vector Graphics (*.svg)"
  4. 在 Inkscape/Illustrator 中编辑
  5. 插入论文

优势:
  ✅ 矢量图，无限缩放
  ✅ 文件小
  ✅ 适合出版
  ✅ 所见即所得
```

---

### 场景 2: 数据分析（原始数据） 🌟

**推荐**: HDF5 Viewer Ctrl+E

```
需求: 导出完整数据到 Excel/MATLAB

步骤:
  1. 选择数据集
  2. Ctrl+E
  3. 选择 CSV
  4. 在 Excel 中打开

优势:
  ✅ 完整分辨率
  ✅ 所有数据点
  ✅ 友好的 CSV 格式
  ✅ 自动提取列名（结构化数组）
```

---

### 场景 3: 高质量位图（存档） 🌟

**推荐**: HDF5 Viewer Ctrl+E

```
需求: 高质量图像，用于存档或进一步处理

步骤:
  1. 选择图像数据
  2. Ctrl+E
  3. 选择 TIFF
  4. 保存

优势:
  ✅ 16-bit 精度（TIFF）
  ✅ 完整分辨率
  ✅ 适合 ImageJ/MATLAB 处理
  ✅ 科研标准格式
```

---

### 场景 4: 快速预览分享 🌟

**推荐**: PyQtGraph 右键导出

```
需求: 快速导出当前视图，分享给同事

步骤:
  1. 调整视图
  2. 右键 → Export → PNG
  3. 发送文件

优势:
  ✅ 快速
  ✅ 所见即所得
  ✅ 适合快速预览
```

---

### 场景 5: 带列名的数据表 🌟

**推荐**: HDF5 Viewer Ctrl+E

```
需求: NeXus 结构化数组导出到 Excel

数据:
  dtype=[('id', 'i4'), ('temp', 'f8'), ('value', 'f8')]

步骤:
  1. 选择结构化数组
  2. Ctrl+E
  3. 保存为 CSV

结果:
  id,temp,value
  1,295.3,100.5
  2,295.7,105.2

优势:
  ✅ 自动提取列名
  ✅ Excel 直接识别
  ✅ 数据整洁
```

## 📖 详细对比

### 1. **导出内容差异**

#### PyQtGraph (右键)
```
导出内容: 当前屏幕显示的视图

示例:
  原始数据: 1000×1000 像素
  屏幕显示: 500×500 像素
  导出 PNG: 500×500 像素 ⚠️
```

#### HDF5 Viewer (Ctrl+E)
```
导出内容: 原始完整数据

示例:
  原始数据: 1000×1000 像素
  屏幕显示: 500×500 像素
  导出 PNG: 1000×1000 像素 ✅
```

---

### 2. **CSV 格式差异**

#### PyQtGraph CSV
```csv
# 基本的 x,y 坐标
0,1.5
1,2.3
2,3.7
```

#### HDF5 Viewer CSV
```csv
# 带列名的完整数据
Col 0,Col 1,Col 2
1,2.5,3.7
4,5.9,6.1

# 或结构化数组
id,temperature,intensity
1,295.3,100.5
2,295.7,105.2
```

---

### 3. **图像质量对比**

| 需求 | PyQtGraph | HDF5 Viewer | 推荐 |
|------|-----------|-------------|------|
| 矢量图 | SVG ✅ | - | PyQtGraph |
| 无损位图 | PNG (视图) | PNG (原始) ⭐ | HDF5 Viewer |
| 高精度 | - | TIFF 16-bit ⭐ | HDF5 Viewer |
| 小文件 | JPEG | JPEG ⭐ | HDF5 Viewer |
| 论文发表 | SVG ⭐ | TIFF | PyQtGraph |

## 🎯 实际工作流程

### 工作流 1: 科研论文准备

```
Step 1: 数据探索
  → 在 HDF5 Viewer 中查看数据
  → 调整视图、颜色映射

Step 2: 导出插图
  → 右键 → Export → SVG (矢量图)
  → 或 Ctrl+E → TIFF (高质量位图)

Step 3: 编辑美化
  → Inkscape 编辑 SVG
  → 或 ImageJ 处理 TIFF

Step 4: 插入论文
  → 直接插入 LaTeX/Word
```

### 工作流 2: 数据分析

```
Step 1: 浏览数据
  → 在 HDF5 Viewer 中查看

Step 2: 导出数据
  → Ctrl+E → CSV
  → 完整数据，带列名

Step 3: 统计分析
  → Excel 打开 CSV
  → 或 Python/MATLAB 读取

Step 4: 制作图表
  → 在分析软件中绘图
```

### 工作流 3: 快速分享

```
Step 1: 调整视图
  → 缩放到感兴趣区域
  → 调整对比度（右键 → Normalization）

Step 2: 快速导出
  → 右键 → Export → PNG

Step 3: 分享
  → 发送给同事
  → 或插入报告
```

## 💡 专业提示

### 提示 1: 组合使用

```
同一个数据集，使用两种方法:

1. PyQtGraph 导出当前视图 (快速预览)
   → 用于演示、讨论

2. HDF5 Viewer 导出原始数据 (完整数据)
   → 用于分析、存档
```

### 提示 2: 格式选择

```
矢量图需求:
  → PyQtGraph → SVG

高精度需求:
  → HDF5 Viewer → TIFF

数据分析:
  → HDF5 Viewer → CSV

快速分享:
  → PyQtGraph → PNG
```

### 提示 3: 视图调整

```
使用 PyQtGraph 右键菜单:

1. Normalization → 调整亮度
2. Histogram → 手动调整范围
3. Export → 导出调整后的视图

然后用 Ctrl+E 导出原始数据
```

## 🔧 快捷操作

### PyQtGraph 快捷方式

| 操作 | 快捷方式 |
|------|----------|
| 导出 | 右键 → Export |
| 重置视图 | Ctrl+0 |
| 归一化 | 右键 → Normalization |
| 直方图 | 右键 → Histogram |

### HDF5 Viewer 快捷方式

| 操作 | 快捷方式 |
|------|----------|
| 导出数据集 | **Ctrl+E** |
| 打开文件 | Ctrl+O |
| 退出 | Ctrl+Q |

## 📊 性能对比

| 数据大小 | PyQtGraph | HDF5 Viewer | 文件大小 |
|----------|-----------|-------------|----------|
| 100×100 | ~1s | ~1s | 相似 |
| 1000×1000 | ~1s (视图) | ~2s (原始) | HDF5 更大 |
| 10000×10000 | ~1s (视图) | ~10s (原始) | HDF5 更大 |

**结论**:
- PyQtGraph 更快（只导出视图）
- HDF5 Viewer 更完整（导出原始数据）

